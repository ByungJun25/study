### 선택문
선택문이란, 어떤 특정 조건에 따라 실행 순서를 변경하도록 제어하는 구문을 말합니다.  
선택문에는 `if`,`if ~ else`, `switch`문이 있습니다.  



### 반복문
반복문이란, 어떤 특정 조건이 만족하는한 계속해서 실행하는 구문을 말합니다.  
반복문에는 `while`, `do ~ while`, `for` 문이 있습니다.
